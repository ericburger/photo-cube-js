PhotoCubeJS: Cube

The PhotoCubeJS jQuery plugin can implement a number of different slideshow animations (options include 'slideshow_random', 'slideshow_reveal', 'slideshow_drop', 'slideshow_rotate', 'slideshow_fade', 'slideshow_shutter', and 'slideshow_up'). The plugin also includes options for adjusting timings and adding play/pause/next/previous buttons. Examples with different options enabled are presented below.

Example One

In the example below, the slideshow uses the 'slideshow_random' animation option. The play/pause button is enabled and loaded as paused.

The HTML for the slideshow is given below. The only requirements are that the div on which the plugin is called (in this case, #1-pcjs_container) contains a list of images and that these images are wrapped in a div with the class pcjs-image. The attributes of an image wrapper div may contain data including a title for the image and a permalink to which the image is linked.

The JS/jQuery code for calling the plugin is given below. The plugin options include width/height (used for setting the aspect ratio of images), reduce (percent of wrapper height and/or width), crop (for cropping the images), navHover (the next/previous buttons), time/delay (for setting the length in milliseconds of animations and delays between animations), and imageLink (for linking the images to the data-permalink). 

  
  jQuery("#1-pcjs_container").PhotoCubeJS({
    imageListClass: "pcjs_image_list_class",
    style: "slideshow",
    special: "slideshow_random",
    width: 1.85,
    height: 1,
    reduce: 70,
    crop: 1,
    navHover: 1,
    time: 1000,
    delay: 5000,
    auto: 1,
    playPause: 1,
    startPaused: 1,
    imageLink: 1,
    postTitle: 0,
    postExcerpt: 0,
  });

Example Two

In the example below, the slideshow uses the 'slideshow_rotate' animation option. A title and excerpt are added to each slide by enabling the postTitle and postExcerpt options and by populating the data-title and data-excerpt parameters of the pcjs-image elements.

The JS/jQuery code for calling the plugin is given below. 

  
  jQuery("#2-pcjs_container").PhotoCubeJS({
    imageListClass: "pcjs_image_list_class",
    style: "slideshow",
    special: "slideshow_rotate",
    width: 1.85,
    height: 1,
    reduce: 70,
    crop: 1,
    navHover: 1,
    time: 1000,
    delay: 5000,
    auto: 1,
    playPause: 0,
    imageLink: 0,
    postTitle: 1,
    postExcerpt: 1,
  });
