PhotoCubeJS

PhotoCubeJS is a jQuery plugin for creating photo slideshows, cubes, and sliceboxes. The plugin implements 2D-3D projections and transformations in JavaScript and renders the images to an HTML5 Canvas (does not use CSS3 transforms). A number of customizable animations for photo transitions are included in the plugin. The code can be found on GitHub and the examples on this page are available under the gh-pages branch.
