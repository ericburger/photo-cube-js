PhotoCubeJS: Cube

The PhotoCubeJS jQuery plugin can implement a number of different cube animations (options include 'cube_swing', 'cube_rot_x', 'cube_fall_in', 'cube_spin', and 'cube_rot_y'). The plugin also includes options for adding a shadow below the cube, adjusting timings, and adding play/pause/next/previous buttons. Examples with different options enabled are presented below.

In the example below, the cube uses the 'cube_swing' animation option. The play/pause button is enabled and loaded as paused.

The only requirements are that the div on which the plugin is called (in this case, #1-pcjs_container) contains a list of images and that these images are wrapped in a div with the class pcjs-image. The attributes of an image wrapper div may contain data including a title for the image and a permalink to which the image is linked.

The JS/jQuery code for calling the plugin is given below. The plugin options include width/height (used for setting the aspect ratio of images), reduce (percent of wrapper height and/or width), shadow (for adding the shadow effect below the cube), crop (for cropping the images to fit to the surface of the cube), navHover (the next/previous buttons), time/delay (for setting the length in milliseconds of animations and delays between animations), imageLink (for linking the images to the data-permalink), and interval (for setting the milliseconds between each frame rendered to the canvas). 

  
  jQuery("#1-pcjs_container").PhotoCubeJS({
    imageListClass: "pcjs_image_list_class",
    style: "cube",
    special: "cube_swing",
    width: 1.85,
    height: 1,
    reduce: 70,
    shadow: 1,
    crop: 1,
    navHover: 1,
    time: 1000,
    delay: 5000,
    auto: 1,
    playPause: 1,
    startPaused: 1,
    imageLink: 1,
    interval: 30,
  });

Example Two

In the example below, the cube uses the 'cube_fall_in' animation option. The navigation and play/pause buttons are disabled and the animations are set to trigger automatically after a 1 second delay.

The JS/jQuery code for calling the plugin is given below. 

  
  jQuery("#2-pcjs_container").PhotoCubeJS({
    imageListClass: "pcjs_image_list_class",
    style: "cube",
    special: "cube_fall_in",
    width: 1.85,
    height: 1,
    reduce: 70,
    shadow: 1,
    crop: 1,
    navHover: 0,
    time: 3000,
    delay: 1000,
    auto: 1,
    playPause: 0,
    startPaused: 0,
    imageLink: 0,
    interval: 60
  });

Example Three

In the example below, the cube uses the 'cube_rot_y' animation. The animations are set to fire only when the previous or next buttons are clicked.

The JS/jQuery code for calling the plugin is given below. 

  
  jQuery("#3-pcjs_container").PhotoCubeJS({
    imageListClass: "pcjs_image_list_class",
    style: "cube",
    special: "cube_rot_y",
    width: 1.85,
    height: 1,
    reduce: 70,
    shadow: 0,
    crop: 1,
    navHover: 1,
    time: 2000,
    delay: 0,
    auto: 0,
    playPause: 0,
    startPaused: 0,
    imageLink: 1,
    interval: 30
  });
